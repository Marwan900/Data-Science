## 📊 Description du projet

Ce projet explore le monde de la **Formule 1** à travers une analyse de données historiques.  
Le but est de comprendre les tendances, performances et facteurs influençant les résultats en F1, à travers un dataset riche en statistiques sur les circuits, les pilotes, les constructeurs et les saisons.

Nous avons utilisé un **dataset Kaggle** sur la Formule 1, couvrant plusieurs années de compétitions, avec des informations sur :

- les pilotes (drivers),
- les écuries (constructors),
- les circuits,
- les classements et résultats,
- les saisons.

---

## 📥 Dataset utilisé

- Source : Kaggle – F1 World Championship Dataset
- Lien : https://www.kaggle.com/datasets/rohanrao/formula-1-world-championship-1950-2020
- Contenu : fichiers CSV relatifs aux pilotes, courses, résultats, constructeurs, circuits, classements...

Les données ont été récupérées via la bibliothèque `kagglehub`.
