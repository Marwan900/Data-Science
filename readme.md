# Projet F1 Data Science

Ce projet analyse et visualise des données liées à la Formule 1 à l'aide de techniques de data science.

## Contenu

- **f1** : Script principal pour le traitement et l'analyse des données F1.

## Prérequis

- Python 3.x
- Pandas, NumPy, Matplotlib, Seaborn (ou autres librairies selon le script)

## Installation

```bash
pip install -r requirements.txt
```
